# Website Catering Delicious

Website bisnis catering sederhana yang dibuat dengan HTML dan CSS.

## Cara Hosting di GitHub Pages

### 1. Buat Repository GitHub
1. Buka [GitHub.com](https://github.com) dan login
2. Klik tombol "New" untuk membuat repository baru
3. Beri nama repository (contoh: `catering-website`)
4. Centang "Add a README file"
5. Klik "Create repository"

### 2. Upload File Website
1. Di halaman repository, klik "uploading an existing file"
2. Drag & drop atau pilih file:
   - `index.html`
   - `style.css`
   - `README.md`
3. Tulis commit message: "Add catering website files"
4. Klik "Commit changes"

### 3. Aktifkan GitHub Pages
1. Di repository, klik tab "Settings"
2. Scroll ke bawah ke bagian "Pages"
3. Di "Source", pilih "Deploy from a branch"
4. Pilih branch "main" dan folder "/ (root)"
5. Klik "Save"

### 4. Akses Website
Setelah beberapa menit, website akan tersedia di:
```
https://[username].github.io/[repository-name]
```

Contoh: `https://johndoe.github.io/catering-website`

## Struktur File
```
web-catering/
├── index.html      # Halaman utama
├── style.css       # File styling
└── README.md       # Dokumentasi
```

## Kustomisasi
Untuk mengubah konten website:
1. Edit file `index.html` untuk mengubah teks
2. Edit file `style.css` untuk mengubah warna/styling
3. Commit perubahan ke GitHub
4. Website akan otomatis terupdate

## Domain Custom (Opsional)
Jika ingin menggunakan domain sendiri:
1. Beli domain dari provider
2. Di repository Settings > Pages
3. Masukkan domain di "Custom domain"
4. Setup DNS di provider domain
